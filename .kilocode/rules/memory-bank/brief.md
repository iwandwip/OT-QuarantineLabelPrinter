# Quarantine Label Printer

A web-based canvas editor for designing quarantine labels, similar to Canva. Users can create custom labels with shapes, text, and images, select paper sizes (A4, F4, etc.), and export as PDF for printing.

## Core Features
- Canvas-based editor with drag-and-drop elements
- Paper size selection (A4, F4, Letter, Legal, A3)
- Basic shapes (rectangle, circle, line)
- Text elements with font customization
- Color picker for fill and stroke
- Undo/Redo functionality
- Export to PDF
- Responsive design

## Technology Stack
- Next.js 15 (App Router)
- TypeScript
- Shadcn UI + Tailwind CSS
- Canvas library: Fabric.js (via fabricjs-react)
- PDF generation: jsPDF

## Target Users
- Warehouse managers
- Quality control personnel
- Manufacturing facilities

## Success Criteria
- Intuitive drag-and-drop interface like Canva
- Support for multiple paper sizes
- High-quality PDF output
- Fast performance
- Mobile-responsive
